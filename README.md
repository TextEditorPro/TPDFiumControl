# TPDFiumControl

Legacy version of the page scrolling PDF control for Delphi.

Latest version for Delphi 13+ https://codeberg.org/TextEditorPro/TPDFiumControl 

## Requires 

Andy's [PdfiumLib](https://github.com/ahausladen/PdfiumLib) core classes - PdfiumCore.pas and PdfiumLib.pas.

## Supports

- AlphaSkins (native) - https://www.alphaskins.com/

## Defines

Define | Description
------ | -----------
ALPHASKINS | Native AlphaSkins support
